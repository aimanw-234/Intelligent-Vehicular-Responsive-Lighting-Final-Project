# Intelligent Vehicular-Responsive Lighting
The Intelligent Vehicular Responsive Lighting System is a smart street lighting project designed to reduce unnecessary energy consumption through automation.
The system uses Arduino as the main controller to manage sensors and lighting operations efficiently.
An LDR sensor is used to detect day and night conditions automatically.
IR sensors are integrated to detect vehicle movement on the road.
During the daytime, all street lights remain OFF to conserve electricity.
At night, the lights operate in dim mode for minimum power usage.
Whenever a vehicle is detected, the corresponding street light increases to full brightness instantly.
The project demonstrates the practical implementation of embedded systems and sensor-based automation.
The system is cost-effective, scalable, and suitable for modern smart city applications.
This project highlights the importance of intelligent technology in improving energy efficiency and automated infrastructure systems.
